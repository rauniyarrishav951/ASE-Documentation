Method EvaluateNumber
Evaluates a mathematical expression to its numeric value.
Syntax
C#
    public static double EvaluateNumber(string expression)
Parameters
expression
The expression to evaluate (literals, variables, or simple operations).
Return Value
The calculated numeric result as a double.
Exceptions
Exception Condition
Exception Thrown for empty expressions, undefined variables, or unsupported syntax.
Remarks
Supports three types of expressions: 1. Numeric literals (e.g., "3.14") 2. Variable references (e.g., "radius") 3. Simple binary operations (e.g., "height - 15")
