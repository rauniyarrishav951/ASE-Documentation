Documentation Preview
Method Set
Parses and validates the array declaration parameters. Extracts type, name, and size from the parameter list.
Syntax
C#
    public override void Set(StoredProgram program, string parameterList)
Parameters
program
The executing program instance (unused in this implementation).
parameterList
A space-separated string containing three parameters: 1. Type ("int" or "real") 2. Array name (identifier) 3. Array size (positive integer) Example: "int numbers 10"
Exceptions
Exception Condition
CommandException Thrown when: - parameterList is null, empty, or whitespace - Parameters do not contain exactly three parts - The size parameter cannot be parsed as an integer - The size parameter is less than or equal to zero

Documentation Preview
Method Execute
Executes the array creation command. Creates an array in the variable store based on the parsed type, name, and size.
Syntax
C#
    public override void Execute()
Exceptions
Exception Condition
CommandException Thrown when: - The array type is not supported (not "int" or "real") - An array with the same name already exists (unless suppressed by existing handling)
Remarks

This method handles duplicate array names gracefully by catching and re-throwing CommandException unless the exception message contains "already exists", which allows for potential future idempotent behavior.

Supported array types: - "int": Creates an integer array - "real": Creates a real number (floating-point) array
