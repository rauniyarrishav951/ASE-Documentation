Class AppCommandFactory
Factory class responsible for creating command instances based on command type strings. Extends the base CommandFactory to provide application-specific command implementations.
Syntax
C#
    public class AppCommandFactory : CommandFactory
Remarks
This factory creates commands that interact with an AppCanvas instance and handles both drawing commands and variable operations specific to the application.

Method MakeCommand
Creates a command instance based on the specified command type string.
Syntax
C#
    public override BooseCommand MakeCommand(string commandType)
Parameters
commandType
The type of command to create (case-insensitive).
Return Value
An instance of a class implementing ICommand.
Exceptions
Exception Condition
CommandException Thrown when: 1. commandType is null, empty, or whitespace 2. commandType is a block command (if, else, end, while) 3. The base factory cannot handle the command type
Remarks
Command type resolution follows this priority order: 1. Block commands (rejected - handled by parser) 2. Variable declarations (int, real) 3. Array operations (array, poke, peek) 4. Output commands (write) 5. Drawing commands (circle, rect, moveto, drawto, etc.) 6. Variable assignments (if commandType matches an existing variable) 7. Base factory implementation (fallback)
